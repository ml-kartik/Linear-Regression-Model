⚙️ How It Works

The model learns the best-fit line using the equation:

y = wx + b

Where:

w = weight (slope)
b = bias (intercept)

Using Gradient Descent, the model updates parameters repeatedly until the error is minimized.

📉 Cost Function

Mean Squared Error is used to measure model performance:

MSE = (1/n) * Σ(y_true - y_pred)²

Lower MSE = Better Predictions ✅
